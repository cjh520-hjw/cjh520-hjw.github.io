#include <iostream>  
using namespace std;  
int main()  
{  
    char c1='a', c2='b', c3='c', c4='\101', c5='\116';  
    cout << c1 << c2 << c3 << '\n';  
    cout << "\t\b" << c4 << '\t' << c5 << '\n';  
    return 0;  
}

答案：

1
2

abc  
       A       N

分析：

  1. 字符变量解码

  • c1='a'、c2='b'、c3='c'：直接对应 ASCII 字符 a、b、c。

  • c4='\101'：八进制转义符转 ASCII 字符 。
    先八进制数 101 转十进制： 按位权展开， \(1×8^2 + 0×8^1 + 1×8^0 = 64 + 0 + 1 = 65\) 。
    再对应 ASCII 字符 A（ASCII 表中十进制 65 代表 A ）。

  • c5='\116'：八进制转义符转 ASCII 字符 。
    先八进制数 116 转十进制： 按位权展开，\(1×8^2 + 1×8^1 + 6×8^0 = 64 + 8 + 6 = 78\) 。
    再对应 ASCII 字符 N（ASCII 表中十进制 78 代表 N ）。

  1. 第二行输出
  • "\t\b"：
    • \t：制表符，使光标跳转到第 8 列
    • \b：退格符，使光标回退到第 7 列。
    • << c4：在第 7 列输出 A（字符 c4）。
    • << '\t'：光标从第 8 列（输出 A 后自动移到第 8 列）跳转到下一个制表位 第 16 列。
    • << c5：在第 16 列输出 N（字符 c5）。
    • 最终效果：第 7 列显示 A，第 16 列显示 N，中间填充 8 个空格（制表位特性，自己画图试一下）。
🫣 制表符 关键是画图（8后一次，8后一次）

Note

  1. 转义字符只代表一个字符，在内存中占一个字节
  2. 编译系统视角：见到\会接着找它后面的字符，把它们处理成一个字符

内存中的字符数据

字符数据以ASCII码存储，所以字符与整型可通用（可赋值可运算）

‘a’——97

‘A’——65

字符型数据和整型数据是可以通用的。但是应注意字符数据只占一个字节，它只能存放 0~255 范围内的整数。

一个字节有8位，每位可以是0或1，所以能表示的不同值的总数是256个（从0计数到255）。

字符串常量

  • 系统会在末尾加上\0作为字符串结束的标志。

    所以"abc"在内存中占3+1=4个字节

    'a'占1个字节，"a"占2个字节

    "abc\n"包含4个字符，5个字节

    "abc\\n"中的\\是合法的转义字符，所以有5个字符（a、b、c、\、n），\\代表\

  • 字符串常量要用字符数组来存放。

符号常量

就是用符号代表常量，如PRICE代表30

#define PRICE 30不是语句，末尾不加分号

在预编译处理时，所有的PRICE被置换为30

符号常量在C中用比较多，在C++中常用常变量

没有类型，内存中没有命名存储单元

好处

  • 见名知意，不提倡使用很多的直接变量
  • 在需要改变一个常量的时候，能够”一改全改“，如商品单价

变量

就是值可以改变的量

因为是通过变量名找到变量值，所以变量名等同地址

变量名

  • 只有字母、数字、下划线，第一个字符必须为字母或下划线
  • 一般变量名用小写字母，常量用大写字母
  • 变量名不能与关键字、系统函数名、类名相同

算术运算符与算术表达式

  • %（模运算符/求余运算符）两边均为整型数据
  • 相除若有一个操作数是负数，则向零取整（取整是向零靠拢的）
  • 加减乘除中若有一个操作数是float型，则运算结果是double型

运算符的优先级与结合性

算术运算符的结合性是左结合性

各类型数据间的混合运算（隐式转换）

char,short -> int

float -> double

int -> long

int -> double

表达式 10 + 'a' + i * f - d / e 的运算次序如下：

  1. 执行 10 + 'a' ：先把字符 'a' 转换为整数 97，二者相加是 107 。
  2. 执行 i * f：把 i 和 f 都转换成 double 类型，结果为 double 型 。
  3. 执行整数 107 与 i * f 结果相加：把整数 107 转成双精度数（形式为 107.000…00 ），再和 i * f 的结果相加，结果是 double 型 。
  4. 执行 d / e ：把变量 e 转换成 double 型，相除后为 double 型 。
  5. 执行前面相加结果与 d / e 结果相减：用 10 + 'a' + i * f 的结果减去 d / e 的结果，最终结果为 double 型 。
